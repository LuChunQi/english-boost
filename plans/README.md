## 一、上海高考英语全面解析

### 1.1 考试结构

| 部分 | 分值 | 时长 | 形式 |
|------|------|------|------|
| 笔试 | 115分 | 105分钟 | 纸笔 |
| 听说测试 | 35分 | 35分钟 | 机考 |
| **总分** | **150分** | **140分钟** | |

**一年两考制度**：1月（春考）+ 6月（秋考），取较高分计入高考成绩。

### 1.2 笔试题型与分值（115分）

| 板块 | 题型 | 题量 | 分值 | 说明 |
|------|------|------|------|------|
| 语法与词汇 | 语法填空 | 10-11题 | 约15分 | 语篇形式，部分给提示词，部分纯填空 |
| 语法与词汇 | 选词填空（十一选十） | 10题 | 约10分 | "小猫钓鱼"，考词性判断+词义辨析 |
| 语言运用 | 完形填空 | 15题 | 约15分 | 四选一，词汇辨析+上下文逻辑 |
| 阅读 | 阅读理解A/B/C篇 | 约15题 | 约30分 | 细节+推理+主旨+词义 |
| 阅读 | 六选四 | 4题 | 约8分 | 句子填入语篇，考衔接连贯 |
| 输出 | 翻译（汉译英） | 4-5句 | 约15分 | **上海卷独有**，从易到难 |
| 输出 | 概要写作 | 1篇 | 约10-12分 | **上海卷独有**，读后写50-60词摘要 |
| 输出 | 作文（Guided Writing） | 1篇 | 约15-20分 | 基于材料/图表/情境的分析性写作 |

### 1.3 听说测试（35分，机考）

| 题型 | 说明 |
|------|------|
| 朗读/模仿朗读 | 给文字，跟读或朗读 |
| 听力理解 | 听对话/短文，选择题 |
| 复述 | 听一段材料，用英语复述要点 |
| 自由表达 | 针对给定话题，发表1分钟观点 |

### 1.4 上海卷 vs 全国卷核心差异

1. **有翻译**（全国卷没有）— 4-5句汉译英，考查语法精确度+词汇灵活度
2. **概要写作替代读后续写** — "提炼+重写" vs "编故事"，完全不同的能力
3. **选词填空"小猫钓鱼"**（全国卷没有）— 考词性+语境
4. **一年两考**（全国卷只有一次）— 战略优势
5. **笔试不含听力**（2025年起）— 笔试纯考读+写+译
6. **作文是 Guided Writing** — 分析性写作，接近雅思，不是模板作文
7. **105分钟笔试** — 时间紧，题量大

### 1.5 140+ 难度分析

| 题型 | 满分 | 140+ 需要 | 容错空间 |
|------|------|-----------|---------|
| 语法填空 | ~15 | 14-15 | 错0-1空 |
| 选词填空 | ~10 | 9-10 | 错0-1空 |
| 完形填空 | ~15 | 13-15 | 错0-2空 |
| 阅读ABC | ~30 | 28-30 | 错0-1题 |
| 六选四 | ~8 | 8 | 错0空 |
| 翻译 | ~15 | 13-15 | 每句仅小瑕疵 |
| 概要写作 | ~12 | 10-12 | 要点全+表达地道 |
| 作文 | ~20 | 17-19 | 高水平完成 |
| 听说测试 | 35 | 33-35 | 近满分 |

**关键洞察**：
- **输入端**（语法+选词+完形+阅读+六选四≈78分）必须稳定在73+，这个靠刷题+语感可以实现
- **输出端**（翻译+概要+作文≈47分）必须拿到40+，这个才是140+的真正门槛
- **听说**（35分）必须33+，不能丢

**总结：对年级前30的学生来说，输入端不是问题，140+的战场在输出端（翻译+概要+作文）和听说测试。**

### 1.6 为什么很多上海学生拿不到140+？

1. **翻译拖后腿** — 语法填空可以满分，但翻译做不到"信达雅"
2. **概要写作不得要领** — 照抄原文降档、要点遗漏、超字数
3. **作文缺深度** — Guided Writing 需要分析+观点+论据，模板不够
4. **时间管理差** — 105分钟做不完，最后写作被压缩
5. **听说测试丢分** — 笔试高分学生在听说丢3-5分，直接140无望
6. **对上海卷特色题型训练不足** — 很多学生用全国卷思路备考上海卷

不再固定"周几做什么"，改为**每周任务包**，孩子可以在一周内灵活安排：

```
📊 每周必做（合计约 3.5 小时）：
├── 翻译训练 ×5组（每组3句，每组15min）= 75min
```

## 六、上海卷五大核心能力专项策略

### 6.1 翻译（汉译英）— 上海卷的"纲"

**核心理念**：翻译练好了，语法填空、选词填空、写作全部受益。

**"先主干后修饰"策略**：
1. 找出中文句子的主语+谓语+宾语
2. 翻译主干（S+V+O）
3. 再补定语、状语、补语

**翻译黄金句型积累**：
- 倒装：Only when... / Not until... / Under no circumstances...
- 强调：It is... that... / do/did/does + 动词原形
- 虚拟：Were I... / Had I known... / Without...
- 从句：what引导的名从、as引导的定从/状从
- 非谓语：Having done... / With + n. + done

**一题多译**：同一个中文句子尝试2-3种英文表达，拓宽思路。

**中国文化表达**：上海卷翻译常涉及中国元素（传统节日、饮食文化、历史典故），需要专门积累。

### 6.2 概要写作 — 上海卷的"分水岭"

**评分标准**：要点齐全 + 用自己的话 + 语言准确 + 50-60词

**四步法**：
1. **读**：划出每段主题句和关键信息
2. **提**：提取3-5个核心要点
3. **换**：同义替换 + 句型转换 + 信息合并
4. **控**：压缩到50-60词

**常见错误**：
- ❌ 照抄原文句子（直接降档）
- ❌ 要点遗漏
- ❌ 超字数
- ❌ 加自己观点（概要只客观总结）

**概要写作核心词汇库**（需专门积累）：
- 表因果：result in / contribute to / lead to / account for
- 表对比：in contrast / whereas / while / on the other hand
- 表递进：furthermore / moreover / in addition / additionally
- 表总结：in conclusion / overall / to sum up / consequently

### 6.3 选词填空（"小猫钓鱼"）

**四步法**：
1. 标词性：拿到11个词立刻标注 n./v./adj./adv. + 时态形式
2. 判空格：冠词后→名词，be后→adj./doing/done，副词→修饰v./adj.
3. 先易后难：确定的先填，缩小范围
4. 代入通读：检查逻辑通顺

### 6.4 作文（Guided Writing）

**和全国卷的区别**：不是模板作文，是分析性写作，更接近雅思。

**三段式结构**：
- 引入段：描述材料/图表核心信息（2-3句）
- 主体段：分析原因+给出观点+论据支撑（5-6句）
- 结论段：总结+建议/展望（2-3句）

**语言提升重点**：
- 高级连接词（避免 always use "I think"）
- 地道表达替换（good→beneficial, bad→detrimental）
- 句式多样性（简单句+复合句+非谓语结构交替）

### 6.5 听说测试

**四部分训练**：

| 部分 | 训练方法 | 频率 |
|------|---------|------|
| 朗读 | 跟读BBC/VOA材料，注意语音语调 | 每天5min |
| 听力 | 精听（逐句听写）+ 泛听（抓大意） | 每天10min |
| 复述 | 听完一段话，用英语复述3-5个要点 | 每周3次 |
| 自由表达 | 高频话题准备+限时表达 | 每周2次 |

**机考适应**：每周至少1次电脑录音模拟，熟悉计时节奏。

**碎片时间利用**：
- 早上起床：听5分钟英文播客
- 晚上睡前：听5分钟英文材料
- 通勤路上：英文歌曲或TED演讲
